欢迎来到我的个人博客！

这是使用 **Astro** + **Cloudflare Pages** 构建的现代化个人博客，采用玻璃拟态（Glassmorphism）设计风格。

## 关于这个博客

这个博客具备以下特点：

- 🚀 **极致性能**：基于 Astro 的零 JS 架构，页面加载飞快
- 💎 **玻璃拟态设计**：半透明面板、背景模糊、渐变色背景
- 🌓 **深色模式**：支持系统偏好和手动切换
- 📱 **响应式布局**：适配手机、平板、桌面端

## 技术栈

| 技术 | 用途 |
|------|------|
| Astro | 静态站点生成 |
| Tailwind CSS | 样式系统 |
| Cloudflare Pages | 托管与部署 |
| Markdown | 内容编写 |

感谢你的访问！
