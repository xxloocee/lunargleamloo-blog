欢迎来到我的个人博客。

这次重构之后，我把它重新整理成了一个更适合长期写作的版本：视觉上采用现在这套 **Neoblo / Neobrutalism** 风格，内容上则尽量围绕技术、工程化、AI 与一些值得认真表达的想法展开。

## 这个博客现在想写什么

我希望这里慢慢长成一个既能放技术文章、也能放思考随笔的小地方，主要会包括：

- **技术与工程实践**：记录真实做过的项目、踩过的坑和实现思路
- **AI 与工作流**：关注 Agent、自动化、软件交付这些正在快速变化的话题
- **一些值得写下来的思考**：论文、知识、生活观察，或者偶尔突然想明白的东西

## 这个博客现在有什么特点

- 🚀 **Astro 驱动**：静态优先，加载轻快
- 🎨 **Neoblo 主题**：粗边框、硬阴影、亮色强调和散落装饰
- 🌓 **深色模式**：支持系统偏好与手动切换
- 🔎 **站内搜索**：支持快速检索已有文章
- 📱 **响应式布局**：适配手机、平板和桌面端

## 技术栈

| 技术 | 用途 |
|------|------|
| Astro | 静态站点生成 |
| Tailwind CSS | 样式系统 |
| Cloudflare Pages | 托管与部署 |
| Markdown | 内容编写 |

如果这些内容刚好也戳中你的兴趣，那就再多逛一会儿。
